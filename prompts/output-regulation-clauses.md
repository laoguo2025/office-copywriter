# Prompt｜Output Mode: Regulation Clauses

## Purpose
把内部规则类文稿收束为“制度条款版”，适用于制度、办法、规定、规范等长期执行文本。

## Use when
- 用户明确说要制度化、办法化、条款化
- 文种属于制度 / 管理办法 / 规定 / 规范

## Output goals
- 条款结构清楚
- 适用范围明确
- 职责分工明确
- 执行流程明确
- 监督与附则完整

## Writing rules
- 优先采用“第一条、第二条……”条款结构
- 常见结构为：目的依据 / 适用范围 / 职责分工 / 管理要求 / 执行监督 / 附则
- 避免纯倡议式、通知式、口号式表达
- 对审批、申领、登记、盘点、问责等动作要写出规则，而非只写原则
- 适合长期执行，不写成一次性工作安排

## Output contract
1. 制度条款版成稿
2. 待补信息（如有）

## Self-check
- 是否是条款体而不是散文体
- 是否能直接作为内部制度底稿
- 是否把规则和流程写清楚
