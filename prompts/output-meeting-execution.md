# Prompt｜Output Mode: Meeting Execution

## Purpose
把会议相关材料收束为“会议落地版”，适用于把会议内容整理成可执行、可跟踪的纪要或行动清单。

## Use when
- 用户要整理会议纪要
- 用户强调会后落实、责任分工、节点推进

## Output goals
- 不追求逐句还原发言
- 聚焦议定事项
- 明确责任人 / 责任部门
- 明确完成时限
- 明确协同与督办要求

## Writing rules
- 结构优先采用“会议基本信息 + 议定事项 + 分工与时限 + 督办要求”
- 每项议定事项尽量写成动作句，而不是讨论句
- 责任主体和时限不能缺
- 若无明确责任人，至少明确责任部门
- 不把纪要写成聊天记录或流水账

## Output contract
1. 会议落地版成稿
2. 待补信息（如有）

## Self-check
- 是否能直接拿去会后执行
- 是否每项任务都有人接、有人跟、有限时
- 是否避免空泛复述会议过程
