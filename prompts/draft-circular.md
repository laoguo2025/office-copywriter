# Prompt｜Draft Circular

## Purpose
把内部典型情况、问题警示、检查结果或奖惩导向，起草为正式通报。

## Use when
- 考勤异常通报
- 安全检查情况通报
- 工作落实情况通报
- 典型问题批评或先进表扬

## Input to infer
- 通报主题
- 涉及对象
- 基本事实
- 问题性质或典型意义
- 处理意见
- 后续要求

## Writing rules
- 标题优先用“关于XXX的通报”
- 必须先把事实写清，再给处理意见或导向要求
- 语气严肃、克制，不情绪化
- 不夸大、不贴标签、不编造处分结果
- 要体现警示、规范或导向价值
- 起草前优先判断主责部门口径，必要时参考：
  - `references/department-routing-rules.md`
  - `references/department-tone-baselines.md`
  - `references/department-vocabulary.md`
- 若通报偏办公秩序、流程执行、报送落实，默认向行政口径收束
- 若通报偏考勤纪律、培训执行、人员管理问题，默认向人事口径收束
- 若通报偏安全检查、现场秩序、设施隐患、保障不到位，默认向后勤口径收束
- 若通报偏重点任务推进、协同落实、阶段检查结果，默认向综合运营口径收束
- 通报要体现部门视角下的警示重点和后续要求，不要只把对象名称替换一下就当作完成差异化

## Output contract
1. 通报成稿
2. 待补信息（如有）

## Self-check
- 是否有事实依据
- 是否有处理意见或导向要求
- 是否适合内部正式留痕
