# usage-scenario-map（内部运营管理场景映射）

## 一、行政场景

- 春节放假安排 → 通知
- 会议召开安排 → 通知
- 培训组织安排 → 通知
- 制度发布 / 修订生效 → 通知 或 制度正文 + 发布通知
- 办公用品采购申请 → 请示
- 对办公用品/设备采购申请作正式审批回复 → 批复
- 固定资产采购申请 → 请示
- 办公用品管理 → 制度 / 管理办法
- 差旅报销管理 → 制度 / 管理办法
- 月度行政工作汇报 → 报告
- 行政专项工作总结 → 总结

## 二、人事场景

- 招聘编制申请 → 请示
- 入职安排通知 → 通知
- 转正 / 调岗 / 任命通知 → 通知 或 决定
- 考勤异常 / 纪律问题 → 通报
- 员工行为规范 → 制度 / 管理办法
- 培训实施安排 → 方案 或 通知
- 人事阶段工作汇报 → 报告
- 人员异常情况说明 → 情况说明

## 三、后勤场景

- 食堂管理通知 → 通知
- 宿舍管理制度 → 制度 / 管理办法
- 车辆管理办法 → 制度 / 管理办法
- 维修 / 保障资源申请 → 请示
- 安全检查情况通报 → 通报
- 消防 / 防疫 / 安全专项安排 → 方案 或 通知
- 后勤保障阶段汇报 → 报告
- 突发后勤异常说明 → 情况说明

## 四、综合运营管理场景

- 跨部门工作推进安排 → 通知
- 重要事项内部部署 → 决定
- 总经理办公会 / 协调会结论 → 会议纪要
- 专项整改方案 → 方案
- 月报 / 季报 / 周报 → 报告 或 总结
- 异常事件内部说明 → 情况说明
- 流程优化 / 制度落地 → 方案 或 制度
- 年度工作计划 / 月度重点计划 → 工作计划
- 年终复盘 / 专项复盘 → 总结

## 一个简单判断法

- 要大家知道并执行 → 通知
- 要上级批准 → 请示
- 要向上汇报 → 报告
- 要沉淀会议议定事项 → 纪要
- 要表扬、批评、传达典型情况 → 通报
- 要规范长期行为 → 制度 / 办法
- 要安排未来工作 → 方案
- 要总结过去工作 → 总结
- 要说明已发生事项与原因 → 情况说明
